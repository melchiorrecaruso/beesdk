The Bee 0.8.0 build 1020 archiver utility, April 2009
(C) 1999-2009 Andrew Filinsky and Melchiorre Caruso

  Usage: Bee <Command> -<Option 1> -<Option N> <ArchiveName> <FileNames...>

  Commands:

    a   Add files to archive
    d   Delete files from archive
    e   Extract files from archive
    x   eXtract files from archive with path name
    l   List archive
    t   Test archive files
    r   Rename files in archive

  Options:

    r       Recurse subdirectories
    u       Update files
    f       Freshen files
    e       force file Extention
    s       create Solid archive
    a       add self-extrActor module
    o<mode> set overwrite file Mode (Q-Query (default), A-All, S-Skip all)
    m<0..3> set compression Method (0-store...1-default...3-maximal)
    d<0..9> set Dictionary size (d1 uses < 5M, d2 (default) < 10M, d3 < 20M...)

    x       eXclude filenames
    t       Test archive after process
    l       List archive after process
    y       set temporany directory
    k       use blowfish crypter/decrypter (min key-length 4 bytes)
    cd<dir> set current archive directory

    cfg<filename> use specified configuration file
    pri<priority> set process Priority (0-Idle, 1-Normal, 2-High, 3-RealTime)

  Use BeeOpt to make most optimal parameters.
